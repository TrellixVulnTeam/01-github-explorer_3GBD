## üöÄ How to run

- Clone the repository
```
git clone https://github.com/onLeoRodrigues/01-github-explorer.git
```
- install dependencies
```
yarn install
```
- Start the server
``` 
yarn dev
```
- Access in your browser
```
http://localhost:8080
```
